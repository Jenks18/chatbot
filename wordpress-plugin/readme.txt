== Installation ==

**Automatic installation (easiest):**

1. Log in to your WordPress admin panel
2. Go to Plugins ‚Üí Add New
3. Search for "ToxicoGPT"
4. Click "Install Now" then "Activate"
5. Go to Settings ‚Üí ToxicoGPT Widget to configure

**Manual installation:**

1. Download the plugin ZIP file
2. Go to Plugins ‚Üí Add New ‚Üí Upload Plugin
3. Choose the ZIP file and click "Install Now"
4. Click "Activate"
5. Go to Settings ‚Üí ToxicoGPT Widget to configure

**FTP installation:**

1. Upload the `toxicogpt-widget` folder to `/wp-content/plugins/`
2. Activate through the 'Plugins' menu in WordPress
3. Configure at Settings ‚Üí ToxicoGPT Widget

== Frequently Asked Questions ==

= Is this chatbot free to use? =

Yes! The widget plugin is completely free. The chatbot service is also free for basic usage.

= Does it store my visitors' conversations? =

The conversations are stored on our secure servers for quality improvement, but no personally identifiable information is collected without consent.

= Can I customize the widget appearance? =

Yes! You can customize:
* Widget position (bottom-right or bottom-left)
* Button color
* Auto-open behavior
* Enable/disable on specific pages

= Is it HIPAA compliant? =

This tool is for educational purposes only and should not be used for actual medical diagnosis or treatment. It is not HIPAA compliant and should not be used to discuss personal health information.

= Can I use it on multiple websites? =

Yes, you can install the plugin on as many WordPress sites as you like.

= How do I disable the widget temporarily? =

Go to Settings ‚Üí ToxicoGPT Widget and uncheck "Enable Widget", then save.

= Does it slow down my website? =

No! The widget loads asynchronously and uses an iframe, so it won't impact your page load speed.
